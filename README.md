# Clustering-and-Dimensionality-reduction

OBJECTIVE
The objective is to evaluate your understanding and ability to apply clustering techniques to a real-world dataset.

Dataset
Used the Iris dataset available in the sklearn library.

Steps Performed

1.Import libraries
2. Load Iris dataset
3. Drop species column
4. Standardize the data
5. Apply PCA (2 components)
6. Elbow Method
7. K-Means Clustering
8. Hierarchical Clustering

Conclusion
Both K-Means and Hierarchical Clustering successfully identified three clusters in the Iris dataset. The elbow method and dendrogram both suggest that three clusters are optimal. PCA enabled clear visualization of these clusters in two dimensions. Although a slight overlap exists between two clusters due to similar flower characteristics, both algorithms effectively grouped similar observations together.
